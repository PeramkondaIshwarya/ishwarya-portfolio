------------------------------
ABOUT THE PROJECT
------------------------------
This is a personal portfolio website designed to showcase my academic and technical background, key projects, skills, and contact information. It is built using HTML, Tailwind CSS.

The portfolio is responsive and optimized for both desktop and mobile viewing. Visitors can explore my recent projects, access my resume, and connect via social platforms.

------------------------------
KEY FEATURES
------------------------------
- Clean and modern user interface
- Responsive design (mobile & desktop)
- Project gallery with images and tech used
- Social media links with icons
- Resume download button
- Contact section

------------------------------
TECHNOLOGIES USED
------------------------------
- HTML5
- Tailwind CSS
- Heroicons / Font Awesome
- Git & GitHub (for deployment)
